# Nail Art Mini Program

## Setup
1. Copy `env.example.js` to `env.js` and edit `BASE_URL`.
2. Open the `miniprogram/` folder in WeChat DevTools.

## Pages
- `pages/extract`: upload finished nail image to extract assets.
- `pages/library`: list assets and navigate to try-on.
- `pages/tryon`: upload bare hand image and generate try-on result.
